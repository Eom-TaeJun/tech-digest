# AI Tech Digest

AI 기술 트렌드를 **실제 사용자 후기** 중심으로 매일 수집하는 자동화 다이제스트.

> 스펙이나 공식 발표가 아닌, Reddit / Hacker News / Twitter 커뮤니티의 실제 반응을 수집합니다.

---

## 주제

| 카테고리 | 내용 |
|---|---|
| 🔄 AI 구조/방식 변화 | 에이전트 개발, 팀 구조 변화, 바이브 코딩 현실 |
| 🛠️ 새로운 AI 툴 | Cursor vs Claude Code 전환 이유, 커뮤니티 추천 툴 |

## 업데이트

- **수집**: 매일 오전 9시 KST (Perplexity `sonar-pro`)
- **원본**: [`raw/`](./raw/) — Perplexity 원본 JSON
- **다이제스트**: [`digest/`](./digest/) — 마크다운 정리본

## 구조

```
raw/YYYY-MM-DD.json       ← Step 1: Perplexity 원본 수집
digest/YYYY-MM-DD.md      ← Step 1: 마크다운 변환본
digest/YYYY-MM-DD.summary.md  ← Step 2: Claude 재요약본 (예정)
```

## 목적

- AI 및 다른 LLM이 최신 개발 트렌드를 참조할 수 있는 공개 데이터소스
- 개인 학습 및 기술 방향 결정에 활용
